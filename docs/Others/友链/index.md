??? tip "如何添加新友链？"
    复制下面的模板，粘贴到 `<div class="friend-links">` 内即可：

    ```html
    <a class="friend-card" href="网站地址" target="_blank">
      <img class="friend-avatar" src="头像链接" alt="名称">
      <div class="friend-info">
        <p class="friend-name">站点名称</p>
        <p class="friend-desc">一句话简介</p>
      </div>
    </a>
    ```

    **头像获取技巧**：GitHub 用户头像可用 `https://avatars.githubusercontent.com/u/用户ID`
